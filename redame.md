# Ruta Arcade Pixel

Juego web arcade estilo pixel art.

## Incluye
- Ruta de 3 carriles
- Árboles al costado
- Puntaje en tiempo real
- Ranking lateral
- Formulario al perder
- Responsive para celular y PC

## Archivos
- index.html
- style.css
- game.js
- api.js

## Cómo probar local
Abrí el proyecto con Live Server en Visual Studio Code.

## Cómo subir a GitHub Pages
1. Crear repositorio en GitHub
2. Subir estos archivos
3. Ir a Settings > Pages
4. Seleccionar la rama principal
5. Guardar

## Backend
Por ahora usa localStorage para pruebas.
Después se conecta a un backend real cambiando `baseUrl` en `api.js`.
